The category is here

$100M+ is flowing into agent memory.

The ex-CEO of GitHub just raised $60M because agent context is broken. He's right. But every player captures what agents do going forward. Nobody mines the institutional knowledge that already exists.

Entire

$60M raised

Founded by GitHub's ex-CEO. Captures forward-looking agent context — prompts, transcripts, checkpoints. Requires teams to adopt a new platform. Context starts fresh.

Mem0

$24M raised

YC-backed memory layer for AI assistants. Series A led by Square Peg with a16z backing. General-purpose — not built for code.

Letta

$10M raised

Felicis-backed stateful agents with long-term memory. Founded by the MemGPT research team. General-purpose agent memory, not code-specific.

Cursor & Cognition

shipping features

Cursor shipped Blame for git context. Cognition (Devin) launched Agent Trace. The tools are converging on memory as a feature, not a product.

Where Codag fits

Every player above captures forward-looking context — what agents do from now on. Codag captures backward-looking context — the years of PR decisions, rejected patterns, and architectural reasoning that shaped your codebase before agents existed. That knowledge doesn't exist in any checkpoint. It only exists if someone mines it. That's us.
